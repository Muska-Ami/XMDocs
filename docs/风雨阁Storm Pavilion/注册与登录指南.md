# 注册

## Java版玩家：

### Step 1

请使用正版，或请参照此处注册红石皮肤站：[红石皮肤站－注册](../红石皮肤站/注册)

### Step 2

加入游戏后，执行命令：
```
/register <密码> <重复密码>
```
请将 `<密码>` 与 `<重复密码>` 更改为您的密码

## 基岩版玩家

加入游戏后，执行命令：
```
/register <密码> <重复密码>
```
或者：
```
/reg <密码> <重复密码>
```
请将 `<密码>` 与 `<重复密码>` 更改为阁下的密码

# 登录

## Java版

### 正版

加入游戏后，执行命令：
```
/login <密码>
```
或者：
```
/l <密码>
```
请将 `<密码>` 更改为阁下的密码

### 红石皮肤站

#### Step 1

请参见：[红石皮肤站－Yggdrasil验证登录](../红石皮肤站/Yggdrasil验证登录)

#### Step 2

加入游戏后，执行命令：
```
/login <密码>
```
或者：
```
/l <密码>
```
请将 `<密码>` 更改为阁下的密码
